## 🖋️ Sobre mim

Sou o tipo de pessoa que acredita que **escrever código e escrever histórias** não são coisas tão diferentes assim.

Nos dois casos, você começa com uma página em branco.  
Nos dois casos, cada palavra (ou linha) importa.  
Nos dois casos, o objetivo é o mesmo: **fazer o outro sentir algo.**

Atualmente estudo pela **Alura**, construindo minha base em Front-end — e em paralelo, também construo narrativas. Porque pra mim, programar é contar histórias com lógica.
